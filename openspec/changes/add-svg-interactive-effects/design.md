## 上下文

当前 about 页面的 BlueprintVsCode SVG 组件缺少交互效果和清晰的标注，导致用户难以理解设计初稿的含义。添加极简标注和交互效果可以提升用户体验，使设计意图更加明确。

## 目标 / 非目标

**目标：**
- 在 Blueprint 区域添加极简标注
- 实现鼠标悬停效果和呼吸外发光
- 确保标注文本在各种状态下保持显著可见
- 在代码区域添加半透明伪代码元素

**非目标：**
- 不改变 SVG 的基本结构
- 不添加复杂的动画效果
- 不修改其他页面的内容

## 决策

1. **标注设计**：
   - 使用极小字号（8-10px）的等宽字体
   - 采用灰色系单色，保持设计初稿的草图美感
   - 确保标注文本在各种状态下保持显著可见

2. **交互效果**：
   - 使用 CSS 动画实现呼吸外发光效果
   - 为每个可交互区域添加透明背景矩形以支持鼠标事件
   - 确保发光效果在正确层级，避免被其他元素遮挡

3. **代码区域处理**：
   - 添加半透明的 `<div>...</div>` 伪代码元素
   - 保持与 SVG 标注内容在视觉风格上一致

## 风险 / 权衡

- **区域检测问题**：SVG 中的 `g` 元素默认不响应鼠标事件，需要添加透明背景矩形或使用 `pointer-events: bounding-box`
- **发光效果层级**：需要确保发光效果在正确层级，避免被其他元素遮挡
- **性能问题**：复杂的动画可能影响页面性能，需保持动画效果简洁
