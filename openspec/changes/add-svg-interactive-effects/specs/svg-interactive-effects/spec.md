## 新增需求

### 需求：Blueprint 区域添加极简标注
系统必须在 Blueprint 区域添加极简标注，包括导航栏框格、侧边栏框格和右侧卡片框格的标注。

#### 场景：导航栏框格标注
- **当** 用户查看 Blueprint 区域
- **那么** 导航栏框格左侧显示微小的"Logo"文字，右侧显示三个水平排列的点作为菜单图标

#### 场景：侧边栏框格标注
- **当** 用户查看 Blueprint 区域
- **那么** 侧边栏框格内纵向等距排列三个短横线，并在下方居中显示"Nav Links"文字标识

#### 场景：右侧卡片框格标注
- **当** 用户查看 Blueprint 区域
- **那么** 第一个卡片框内居中显示"Hero Section"字样，第二个卡片框内居中显示"Feature List"字样

### 需求：实现交互效果
系统必须为所有标注区域添加鼠标悬停效果，当鼠标移动到对应区域时，该区域产生微弱的呼吸外发光效果。

#### 场景：鼠标悬停效果
- **当** 用户将鼠标移动到标注区域
- **那么** 该区域产生微弱的呼吸外发光效果

#### 场景：标注文本可见性
- **当** 用户查看标注区域
- **那么** 标注文本在各种状态下保持显著可见

### 需求：代码区域处理
系统必须在右侧代码框内添加半透明的 `<div>...</div>` 伪代码元素作为视觉呼应，保持与 SVG 标注内容在视觉风格上一致。

#### 场景：代码区域伪代码
- **当** 用户查看代码区域
- **那么** 代码区域显示半透明的 `<div>...</div>` 伪代码元素
